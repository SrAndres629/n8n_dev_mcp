Create a `.env` file with:
```env
N8N_BASE_URL=http://localhost:5678
N8N_API_KEY=your_api_key
ENABLE_DOCKER_TOOLS=True
ENABLE_N8N_TOOLS=True
```

### 3. Launching
```bash
# Production (SSE via FastAPI)

# CLI / Debug Mode
python run_stdio.py
```
